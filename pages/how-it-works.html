How It Works

How Baymax Works

Baymax is the AI chatbot on this site. It answers questions about my background using a curated knowledge base and a small LangGraph pipeline.

The answer pipeline

Each question flows through a LangGraph pipeline. Self-critique is enabled, so the chatbot reviews and can revise its own answer before replying.

retrieve

Finds the most relevant sections from the knowledge base using keyword matching.

keyword based

generate

Produces an answer based on the retrieved context.

llama-3.3-70b-versatile

check

Reviews its own answer for completeness and accuracy (self-critique).

llama-3.3-70b-versatile

revise

If the check finds problems, rewrites the answer once to fix them.

Behind it

Orchestration LangGraph
Inference Groq (llama-3.3-70b-versatile)
Caching Upstash Redis (24h), with per-IP rate limiting
Hosting Vercel serverless

Answer quality is tracked separately on the model evaluation dashboard.
